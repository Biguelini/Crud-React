#### Como executar o projeto
```
  git clone https://github.com/Biguelini/Crud-React.git
  
  cd Crud-React
  
  cd backend
  
  Execute o comando yarn start
  
  Abra um novo terminal na pasta Crud-React
  
  cd frontend
  
  Execute o comando yarn start
```
